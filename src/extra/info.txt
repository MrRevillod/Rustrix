rank:

    Primero, la función verifica si el rango ya ha sido calculado previamente.
    Si es así, devuelve el valor almacenado.

    Luego, se crea una copia de la matriz original para realizar las
    operaciones de fila sin modificar la matriz original.

    Si la matriz tiene solo una fila o una columna, el rango se calcula
    contando el número de elementos no nulos.

    Para matrices con más de una fila y columna, se inicializa el rango
    en 0 y se itera a través de las filas de la matriz.

    En cada iteración, se busca un pivote no nulo en la columna actual.
    Si el pivote es 0, se intercambian las filas para obtener un pivote no nulo.

    Si se encuentra un pivote no nulo, se realizan operaciones de fila
    para hacer ceros debajo del pivote en la columna actual.

    El rango se incrementa en 1 y se repite el proceso hasta que se hayan
    procesado todas las filas o se alcance el límite de las columnas.

    Finalmente, se almacena el rango calculado y se devuelve.
